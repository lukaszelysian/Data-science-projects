# Notebooks

This directory contains Jupyter notebooks for exploratory data analysis and experimentation.

## How to Use

```bash
jupyter notebook
```

Notebooks in this directory are for:
- Data exploration
- Quick prototyping
- Analysis workflows
- Visualization demonstrations

## Naming Convention

- `01_data_loading.ipynb`
- `02_eda.ipynb`
- `03_model_training.ipynb`
