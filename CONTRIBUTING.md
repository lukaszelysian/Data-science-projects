# Contributing

Thank you for your interest in this project!

## Guidelines

1. **Fork the repository** and create a new branch for your work
2. **Commit directly to `main`** for straightforward updates
3. **Use clear commit messages** describing what was changed
4. **Follow the existing code style** and directory structure
5. **Update documentation** if you add new features

## Commit Message Format

```
Type: Brief description (50 chars max)

Detailed explanation (if needed)
```

**Types:**
- `feat:` New feature or project
- `fix:` Bug fix
- `docs:` Documentation
- `refactor:` Code refactoring
- `style:` Formatting changes
- `data:` Data updates

## Direct Upload Process

This repository uses direct commits to `main` for simplicity:

```bash
git clone https://github.com/lukaszelysian/Data-science-projects.git
cd Data-science-projects
git checkout main
# Make your changes
git add .
git commit -m "type: Your clear message"
git push origin main
```

No pull requests needed—just commit directly!
